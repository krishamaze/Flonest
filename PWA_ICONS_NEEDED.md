# PWA Icons Needed

The following icon files are required in the `public/` directory:

## Required Files
1. **pwa-192x192.png** - 192x192 pixels
2. **pwa-512x512.png** - 512x512 pixels

## Design Guidelines
- Use the brand primary color: **#E2C33D** (yellow/gold)
- Background: white or transparent
- Simple, recognizable icon
- Works well when scaled down
- Follows your brand identity

## Icon Content Suggestions
- **Option 1**: Letter "F" for FineTune (bold, modern font)
- **Option 2**: Simple barcode/inventory icon
- **Option 3**: Shopping cart or box icon
- **Option 4**: Abstract geometric shape with brand colors

## Tools to Create Icons
- Figma / Adobe Illustrator (design)
- [RealFaviconGenerator.net](https://realfavicongenerator.net/) (generate all sizes)
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
